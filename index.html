TradeListView.js

            
  var template = Handlebars.compile($('#tmpl-trades-item').html());

  var html = template({
    id: trade.id,
    pair: trade.get('pair')
  });

  $('.trades-list').append(html);

TradeListView.js

            
  var html = Handlebars.template.trades-item({
    id: trade.id,
    pair: trade.get('pair')
  });

  $('.trades-list').append(html);

Benefits

  1. maintainable (isolated, organized, logic-less)
  2. smaller templating library (runtime only)
  3. faster execution (compiled at build time)
