Gerrit and Github

Face-off

Gerrit

(+) Remote git repository, automatically keeps in sync with upstream

(-) Austere UI, 1 review per commit

Github

(+) Very popular , 1 review per branch

(-) Manually keep fork in sync with upstream, lots of merge clouds history
